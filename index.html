Projets

TeleTrack — Application web de gestion du télétravail

Spring Boot • Java • HTML • CSS • JavaScript • H2

Conception et développement d’une application web dédiée à la gestion des demandes de télétravail au sein de PORTNET. La plateforme permet aux employés de soumettre leurs demandes et aux gestionnaires de les consulter, valider ou rejeter via une interface claire et sécurisée.

  • Mise en place d’un système d’authentification sécurisé avec gestion des rôles employé / manager
  • Développement d’un backend Java avec Spring Boot et exposition d’API REST
  • Conception d’interfaces web responsives pour la gestion, le suivi et la visualisation des demandes
  • Intégration d’un calendrier interactif et import des employés via fichier CSV

Système de microfinance — Oracle Database & Laravel

Oracle • SQL • PL/SQL • Laravel • MVC

Conception et implémentation d’un système de gestion de microfinance reposant sur Oracle Database et une interface web Laravel. Le projet met l’accent sur la sécurité, l’administration de la base de données, l’automatisation des traitements et la structuration des accès.

  • Conception d’une base de données relationnelle sécurisée pour la gestion des clients, prêts, échéances et remboursements
  • Gestion des utilisateurs, rôles et privilèges pour renforcer le contrôle d’accès aux données sensibles
  • Développement de procédures stockées, fonctions et triggers en PL/SQL pour automatiser les règles métier
  • Création d’une interface web Laravel en architecture MVC pour interagir avec la base Oracle

Carrefour Sales Analysis — Power BI Dashboard

Python, Power BI, DAX

Projet complet de Business Intelligence basé sur les ventes Carrefour. Pipeline ETL Python, modélisation en schéma étoile et dashboards interactifs Power BI pour analyser les performances commerciales.

  • Préparation et transformation des données de ventes via un pipeline ETL Python utilisant Pandas
  • Modélisation du dataset en schéma étoile afin d’optimiser les performances analytiques dans Power BI
  • Création de mesures DAX et de KPIs pour l’analyse du chiffre d’affaires, des remises et des annulations
  • Conception de dashboards interactifs permettant l’analyse des ventes par catégorie, géographie, période et comportement client
